# Architecture Decision Records

One page per significant decision. Copy the shape below into
`NNNN-short-title.md`, keep it under a page, and never rewrite history — a
reversed decision gets a new ADR that supersedes the old one.

```markdown
# ADR-NNNN: Title

Date: YYYY-MM-DD · Status: Proposed | Accepted | Superseded by ADR-NNNN

## Context
What forces were at play; what problem demanded a decision.

## Decision
What we chose, stated as a rule the codebase follows.

## Consequences
What becomes easier, what becomes harder, what it commits us to.
```
